# NLP Emotion Classifier

Zero-shot emotion classification on tweets using LLMs via OpenRouter.

## Setup
1. `pip install -r requirements.txt`
2. Copiar `.env.example` para `.env` e preencher as variáveis
3. `python main.py`

## Modelos testados
- `meta-llama/llama-3.1-8b-instruct`
- `mistralai/mistral-7b-instruct`

## Emoções
sadness · joy · love · anger · fear
